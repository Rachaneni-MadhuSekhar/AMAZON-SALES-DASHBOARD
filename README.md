## Project Overview
This dashboard provides a comprehensive view of sales performance, profitability, and customer distribution. It helps stakeholders make data-driven decisions by visualizing trends and patterns in the dataset.

 ## Key Features
-Total Sales, Profit, Cost & Quantity KPIs
-Sales by Category (Technology, Furniture, Office Supplies)
-Sales Trend Analysis (Year & Month)
-Profit by Region (West, East, Central, South)
-Interactive Filters:

## Dashboard Insights
-Technology category contributes the highest sales.
-Sales show a fluctuating but overall increasing trend over time.
-Certain regions (like West/East) generate higher profit compared to others.
-Customer base and purchasing patterns vary across segments.

## Tools & Technologies
-Power BI
-Data Modeling
-DAX (Data Analysis Expressions)
-Data Cleaning & Transformation

## How to Use
-Download the .pbix file from this repository
-Open it in Power BI Desktop
-Interact with filters and visuals to explore insights
